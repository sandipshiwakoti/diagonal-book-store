# Diagonal Book Store

Diagonal Book Store is an online book store which helps you buy IT,Programming and Computer Science Books. It offers a wide range of popular books.

Tech Stack:

- React JS
- Tailwind CSS
- Material UI
- Redux
- Redux Thunk
- React Router
- React Icons
- React Toastify
- React Testing Library
